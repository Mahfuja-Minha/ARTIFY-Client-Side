##  About ARTIFY

ARTIFY is a modern online art gallery platform where artists can showcase their creativity and users can explore, like, and save artworks. The platform focuses on clean UI, smooth user experience, and community engagement.

---

##  Key Features

-  Artists can add, update, and manage their artworks through a personal gallery.
-  Users can explore public artworks with detailed views.
-  Favorite system allows users to save artworks they love.
-  Light & Dark theme toggle for better user experience.
-  Secure authentication system with personalized user access.
-  Fully responsive design for mobile, tablet, and desktop devices.
